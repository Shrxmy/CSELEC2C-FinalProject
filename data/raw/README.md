# Raw Data Metadata

This folder is reserved for local copies of the Global Terrorism Database (GTD).

Expected files for this project:

- `globalterrorismdb_0522dist.xlsx`: GTD distribution covering 1970-2022.
- `globalterrorismdb_2021Jan-June_1222dist.xlsx`: GTD distribution for January-June 2021.

Source:

- START Global Terrorism Database: https://www.start.umd.edu/data-tools/GTD
- Kaggle mirror: https://www.kaggle.com/datasets/START-UMD/gtd

The raw data files are intentionally ignored by Git because they are large distribution files.
To reproduce the analysis, download the GTD file from START or Kaggle and place it in this
folder with the expected filename.
